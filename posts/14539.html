蓝桥云课实验楼linux打基础

  • who 显示已登录用户的信息.

    • who -a (-all 显示目前用户的详细信息)
    • who -q (列出所有登录用户登录名, 用户数量)
    • who -u (列出已登录的用户)
    • who -r (查看当前登录用户级别)
    • who -b (上次系统启动时间)

  • top 显示当前系统正在执行的进程相关信息

    • top -p 1138 (显示进程号为1138的进程信息)
    • top -Hp 1235 (显示该进程下的各个线程运行情况)

  • vim删除所有内容.

    ggdG: gg,跳转到文件首行, d为删除, G为跳转到文件末尾行.

shell 常用通配符

字符含义
*匹配0或多个字符
?匹配任意一个字符
[list]匹配list中的任意单一字符
[^list]匹配除list以外的任意字符
[c1-c2]匹配c1-c2中的任意单一字符, 如:[0-9] [a-z]
{string1, string2, …}匹配string1 或 string2 (更多)其一字符串
{c1..c2}匹配c1-c2中全部字符 如{1..10}

常用快捷键作用
ctrl+d键盘输入结束或退出终端
ctrl+s暂停当前程序, 然后按任意键恢复.
ctrl+z将当前程序放到后台运行, 恢复到前台为命令 fg.
ctrl+a将光标移动至行头, 相当于 Home 键.
ctrl+e将光标移动至行尾, 相当于 End 键.
ctrl+k删除从光标所在位置到行末.
alt+backspace向前删除一个单词.
shift+PgUp终端翻页 上
shift+PgDn终端翻页 下

ls -l

查看文件权限

  • FHS（英文：Filesystem Hierarchy Standard 中文：文件系统层次结构标准），多数 Linux 版本采用这种文件组织形式，FHS 定义了系统中每个区域的用途、所需要的最小构成的文件和目录同时还给出了例外处理与矛盾处理。

  • cp 复制, rm删除. 要操作一个目录, 加上 -r或**-R**参数.
  • 遇上删除权限不足 -f 强制删除.
  • rm -rf /*✔️
  • mv 源目录文件 目的目录
  • mv 还能重命名文件
  • mv file1 file2 (将文件名file1重命名为file2).
  • 批量重命名, mv就力不从心了.
  • rename是专业户(不是内置命令, 需要安装), 再配上正则表达式使用.
  • cat, tac, nl命令查看文件
  • cat为正序显示, tac为倒序显示.
  • 在 Linux 中文件的类型不是根据文件后缀来判断的。
  • file 查看文件类型, 查看一个空的.txt文件会输出 empty.
  • declare 对shell环境变量进行显示,设置.

  • $符号用于表示引用的一个变量的值.

  • 变量类型
    • 当前 Shell 进程私有用户自定义变量，如上面我们创建的 tmp 变量，只在当前 Shell 中有效
    • shell本身内建的变量
    • 从自定义变量导出的环境变量
  • 与环境变量相关的命令
命令说明
set显示当前shell的所有变量,包括其内建环境变量(与shell外观等相关),用户自定义变量及导出的环境变量
env显示与当前用户相关的环境变量, 还可以让命令再指定环境中运行
export显示从shell中导出成环境变量的变量, 也能通过它将自定义变量导出为环境变量

linux变量生存周期实现方式
永久的需要修改配置文件,变量永久生效
临时的使用export命令行声明即可,变量在关闭shell时失效

  • /etc/bashrc (有的linux没有) : 存放shell变量

  • /etc/profile : 存放环境变量(永久生效的)

文件前缀为 **.**的即为隐藏文件.

  • shell通过环境变量PATH来进行搜索命令然后执行.

  • 创建简单的shell脚本

创建脚本hello_shell.sh

#!/bin/bash
for ((i=0; i<10; i++));do
  echo "hello shell"
done
exit 0

添加可执行权限.

chmod 755 hello_shell.sh

执行.

  • 创建一个c语言hello world程序:

创建文件hello_world.c文件.

#include <stdio.h>

int main(void)
{
    printf("hello world!\n");
    return 0;
}

保存后使用gcc生成可执行文件.

gcc -o hello_world hello_world.c

直接运行即可.

  • ​ >> 表示将标准输出以追加的方式重定向到一个文件中, 注意前面用到的 **> **是以覆盖的方式重定向到一个文件中, 使用的时候一定要注意分辨. 在指定文件不存在的情况下都会创建新的文件.

  • 变量的修改

变量设置方式说明
${变量名#匹配字串}从头向后开始匹配, 删除符合匹配字串的最短数据
${变量名##匹配字串}从头向后开始匹配, 删除符合匹配字串的最长数据
${变量名%匹配字串}从尾向前开始匹配, 删除符合匹配字串的最短数据
${变量名%%匹配字串}从尾向前开始匹配, 删除符合匹配字串的最长数据
${变量名/旧的字串/新的字串}将符合旧字串的第一个字串替换为新的字串
${变量名//旧的字串/新的字串}将符合旧字串的全部字串替换为新的字串

搜索文件

  • whereis 简单快速

从数据库中查询, 搜索二进制文件(-b), man帮助文件(-m)和源代码文件(-s)

  • locate 快而全

非内置命令. 通过查询 /var/lib/mlocate/mlocate.db 数据库来检索信息.

系统会使用定时任务每天自动执行 updatedb命令来更新数据库.

locate /etc/sh

他会查找 /etc 下所有以sh开头的文件 (不只在 /etc 目录下, 还会递归子目录查找)

只统计数目加上 -c 参数, -i参数忽略大小写进行查找.

  • which 小而精

内建命令, 通常用来确定是否安装了某个指定的程序, 因为它只从PATH环境变量指定的路径中去搜索命令并返回第一个搜索到的结果.

  • find 精而细

它不但可以通过文件类型、文件名进行查找而且可以根据文件的属性（如文件的时间戳，文件的权限等）进行搜索。

find常见格式

sudo  find   /etc/  -name  interfaces

去/etc/目录下, 搜索名字叫做 interfaces 的文件或目录.

与时间相关的命令参数说明
-atime最后访问时间
-ctime最后修改文件内容的时间
-mtime最后修改文件属性的时间
… n在n天之前的”一天之内”
… +n在n天之前(不包括n天本身)
… -n在n天之内(包括n天本身)
-newer filefile为以一个已存在的文件,列出比file还要新的文件名

find [path] [option] [action].
