安装Ubuntu20虚拟机

  1. 准备工作, 下载虚拟机镜像.
    • 🐖①: 如果浏览器下载速度很慢, 建议使用迅雷下载, 浏览器扩展里找迅雷下载支持的插件.
  2. 打开vmware, 选择 创建新的虚拟机.
  3. 选择自定义.
  4. 下一步.
  5. 稍后安装操作系统.
  6. 下一步.🐛🐛🐛
  7. 起个名字, 自定义个位置就好.
  8. 这个默认就行.
  9. 我推荐的4G.🤔
  10. 这个还是默认.🐷
  11. 推荐的, 下一步.
  12. 创建虚拟盘.
  13. 分配空间, 我比较拮据, 选25GB, 这个自由调控.🦄🦄🦄
  14. 下一步.
  15. 自定义硬件.
  16. 选择iso文件, 打印机可以移除.
  17. 然后点击完成, 开启此虚拟机.
  18. 选中文, 然后点安装.
  19. 下一步就好….
  20. 设置用户名和密码
  21. 然后就等着, 比较慢.🐛🐛🐛
  22. 如果显示屏幕很小, 在ubuntu设置里调一下分辨率就行了.🦒🦒🦒
  23. 经历漫长的等待, 重启虚拟机, 安装成功.✔️✔️✔️

安装一个好康的😲😲😲

Ctrl+Alt+T打开终端.

sudo apt-get install cmatrix

这个命令用来安装东西, 先不用理解. 跟着敲一下就完了🐵🐵🐵

cmatrix

运行一下, 看看会发生什么.🦒🦒🦒

终端里停止正在运行的命令 通常使用快捷键 Ctrl+C.

输入 clear 清屏.

一个小任务🦍🦍🦍

在终端里使用vim编辑写一个c语言的代码, 比如打印 Hello, world, 编译并运行.

如果没有vim, 先运行 sudo apt-get install vim安装一下.

vim hello.c                    # 注释: 这里我们使用vim打开一个新文件hello.c

什么是vim?

​ Vim 是从 vi 发展出来的一个文本编辑器。代码补全、编译及错误跳转等方便编程的功能特别丰富，在程序员中被广泛使用。简单的来说， vi 是老式的字处理器，不过功能已经很齐全了，但是还是有可以进步的地方。 vim 则可以说是程序开发者的一项很好用的工具。

菜鸟教程-vim, 了解一下就行, 以后用到的多了也就会了.(我还不会)🤥🤥🤥

  • 先 敲入 i 进入编辑模式, 然后就能敲 c 的代码了.

  • 敲完怎么退出? 先ESC, 然后输入 :wq(保存并退出).

  • 这时敲个 ls 查看当前目录下的文件.

  • 代码有了, 要编译一下.

    gcc -o hello hello.c  # 注释: 相当于使用gcc编译器编译生成可执行文件 hello
                    # 没有gcc 那就安装一下.

    • 可以看到, 出现了个hello文件, 绿色代表有可执行权限.

      ./hello			# 运行hello, ./是当前路径

​ 在Linux中, 绝对路径一定是由 **根目录 /**开始写起的, 用 . 表示当前目录, 用 .. 表示 上一级(父目录).

使用 pwd 可以打印当前路径, cd 跳转到指定路径.

待办任务💯💯💯

  1. 试着编写一个c语言代码, 输入数字n(1-9), 打印九九乘法表的前n行. 写完在终端编译运行.🤔🤔🤔
  2. python作业, 注: 没有python环境就装一个,很快.😏

  • 使用 touch + 名称.py创建一个python文件.

  • 使用 vim 打开编辑刚刚创建的文件.

    #!/usr/bin/env python

# 写下python代码, 比如最简单的打印 Hello, world!

  • 使用 ./demo.py , 即运行我写的demo.py

  • 运行会出错, 使用 ls查看.

    • 可以看到 demo.py 不是绿色

    • 在输入ls -l康康.

    • 缺少可执行权限.

      chmod +x demo.py		# 添加可执行权限

  • 再次尝试运行这个py文件.

最后, 终端输入 poweroff 关机.🥱🥱🥱
